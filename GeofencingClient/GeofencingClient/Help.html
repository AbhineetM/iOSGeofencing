Geo-Fencing

Geofencing allows us to create a virtual geographical boundry, inside which an 'Observer' device can monitor movements of other selected devices('Subjects'). If any of the subject device crosses the boundry the observer is notified.


Modules

[A] Tracker

Tracker is the primary feature of the app. Here user selects his/her role as an 'Observer' or a 'Subject'. Observer is the user who is tracking the location movements and Subject is the user who is being tracked. Oserver also sets the boundry radius marking the fence for monitoring.

1. Monitor
Use this option to select the role of an 'Observer'.

2. Get Monitored
This option is to select the role of a 'Subject'. An observer is only able to pick subjects who have registered with this option. Also, a 'Subject' can simultaneously be an 'Observer' for other subjects and vise-versa.

3. Who Am I Monitoring
If the user is an observer than this option displays all the subjects that he/she is monitoring.

4. My Info
This displays the current GPS location of the device.

[B] Point of Interest

This uses 'Augmented - Reality' that enables you to visualize Points of Interest (POI) located in your nearby environment, using the iPhone camera. POI database includes banks, education institutes, health centers, hotels etc

[C] Routes

Routes can be used by an 'Observer' 'to check the past movements of his subjects. The path of movement(route) is plotted on the map making it easy to see the movements.

Routes history is available of each day, where the observer can check each day's movements for a particular subject.

[D] Speed

Speed displays the current speed of the device. Its a useful feature when you are on the move when using this app.
